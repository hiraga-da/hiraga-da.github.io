# 平賀文書の整理

平賀譲（1878-1943）に関する研究・資料アーカイブサイトです。

## 概要

平賀譲は、戦艦大和をはじめとする旧日本海軍艦艇設計の中心人物であり、東京帝国大学総長として大学の自治を守り抜いた技術者・教育者です。

本サイトでは、平賀譲の略歴、事蹟、関連研究、関連書籍などを紹介しています。

## コンテンツ

- **トップページ** - サイト概要
- **略歴** - 平賀譲の生涯
- **事蹟**
  - 総長としての事蹟
  - 軍人としての事蹟
- **メンバーの研究** - 研究論文・発表
- **五月祭展示講演会報告** - 東京大学五月祭での展示活動
- **リンク・関連書籍** - 参考資料
